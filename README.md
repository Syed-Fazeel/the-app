# The Apps Developers — Revamp (Next.js)

This workspace is a minimal Next.js scaffold to start revamping theappsdevelopers.com with design concepts inspired by artemez.co. It contains a basic layout, hero, and a placeholder HEOR section.

Getting started:

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site.

Next steps:

- Add images, brand assets, and refine styles
